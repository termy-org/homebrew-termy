# homebrew-termy

[![Homebrew Tap](https://img.shields.io/badge/Homebrew-Tap-FF9E0F?logo=homebrew)](https://github.com/termy-org/homebrew-termy)
[![Latest Release](https://img.shields.io/github/v/release/termy-org/termy?color=blue&label=latest)](https://github.com/termy-org/termy/releases)

Homebrew tap for [Termy](https://termy.dev) - A minimal GPU-powered terminal written in Rust.

## Installation

```bash
# Add the tap
brew tap termy-org/termy

# Install Termy
brew install --cask termy
```

Or in one line:

```bash
brew install --cask termy-org/termy/termy
```

## Upgrading

```bash
brew upgrade --cask termy
```

## Uninstalling

```bash
brew uninstall --cask termy
brew untap termy-org/termy
```

## About Termy

Termy is a minimal, GPU-powered terminal emulator built with Rust and GPUI.

- 🚀 GPU-accelerated rendering
- 🎨 Customizable themes
- ⌨️ Powerful keybindings
- 🖥️ Cross-platform (macOS, Windows, Linux)

Visit [termy.dev](https://termy.dev) for more information.
